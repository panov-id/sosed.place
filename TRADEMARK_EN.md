# Trademark & Name Protection — Starter Template

> Draft checklist. Fill in as you go. This is not legal advice — verify current fees and
> rules with an attorney or the official office website before filing.

## 0. What we can actually protect

- **The idea — no.** Only the form of expression is protected, not the concept.
- **Name and logo — a trademark (™ / ®).** The main and cheapest lever.
- **Code, texts, design — copyright.** Arises automatically, no registration required.
- **Domains — separate** (registrar), but worth securing early.

Startup priority: **first the name/logo as a mark, everything else as you grow.**

---

## 1. "Do it now" checklist (free / cheap)

- [ ] Lock the final **name** and its spelling (sosed.place / Sosed / ...).
- [ ] Check availability: domains (.place, .com, .app), social handles, app stores.
- [ ] **Preliminary search** in trademark databases — nothing similar taken (see §4).
- [ ] Collect and store **logo** sources (vector) and creation date.
- [ ] Start using the **™** symbol next to the name now (allowed without registration).
- [ ] Decide the owner: **individual / sole proprietor / company** (a legal entity is preferable).

---

## 2. What exactly gets registered

| Object | What it gives | Comment |
|---|---|---|
| Word mark (the name itself) | Protects the name in any styling | Most important. Start here. |
| Figurative mark (logo) | Protects the specific image | Pricier, narrow protection |
| Combined (name + logo) | Protects the pairing | A compromise |

**Recommendation:** prioritize the **word mark** (the name). Logos change, names don't.

---

## 3. Nice Classification classes

A mark is registered per **classes of goods/services**, not "in general". For a social network:

- [ ] **Class 9** — software, mobile apps, downloadable software.
- [ ] **Class 38** — telecommunications, access to chats/forums, message transmission.
- [ ] **Class 42** — SaaS, software development and maintenance, platform hosting.
- [ ] **Class 45** — online social networking services, dating/community (often key for a social app).
- [ ] (opt.) **Class 35** — advertising, if ad monetization is planned.

> More classes = higher cost. Take the minimum needed: usually **9 + 42 + 45**.

---

## 4. Preliminary search (before filing)

- [ ] **EUIPO eSearch plus / TMview** — EU and worldwide marks: https://www.tmdn.org (TMview).
- [ ] **WIPO Global Brand Database** — international marks: https://branddb.wipo.int
- [ ] National databases of target countries.
- [ ] Plain web search + app stores (whether it's already taken in practice).

Goal — confirm there is no **identical or confusingly similar** mark in your classes.

---

## 5. EU and international registration

### Option A — Whole EU (EUIPO)
- One application = protection across **all EU countries**.
- Office: **EUIPO** (euipo.europa.eu).
- Pro: single EU-wide coverage in one filing.
- Con: "all or nothing" — an opposition in one country can sink the whole application.

### Option B — International (Madrid System, WIPO)
- File **one international application** via WIPO and pick member countries.
- **Requires a "base" mark** — first a registration/application in your "home" country; the international one builds on it.
- Handy when you need specific countries outside the EU (US, UK, etc.).
- Site: wipo.int/madrid.

### How startups usually do it
1. Register a **base mark at home** (country where the business is registered).
2. **Extend** to target markets via the Madrid System (EU/US/UK/…).

> Verify exact fees and timelines on the EUIPO / WIPO sites at filing time — they change.

## 6. Roadmap

1. [ ] Finalize name and logo.
2. [ ] Run the preliminary search (§4).
3. [ ] Choose Nice classes (§3).
4. [ ] Choose the coverage strategy (§5): home → EU/international.
5. [ ] Decide: file yourself or via a **trademark attorney** (EU/international — usually via an attorney).
6. [ ] File the application, pay the fees.
7. [ ] Pass examination and the opposition period.
8. [ ] Receive the certificate → use **®**.
9. [ ] Set a **renewal** reminder (typically every 10 years).

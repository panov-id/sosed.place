# Screen 16 — Referral Program (Beyond the Alpha)

## Purpose

The mechanic for inviting new users with a reward — part of the internal balance (see the README's "Beyond the alpha" section). Not part of day0/the alpha.

## Logic

- Each user has a personal referral link.
- When an invitee follows the link and registers, both sides get a balance reward — the inviter and the invitee.

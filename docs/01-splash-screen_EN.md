# Screen 1 — Loading and Birth Year Selection

## Purpose

The first screen a user sees when opening sosed.place. Combines the loading animation with the first registration step — picking a birth year.

## Visual

An animation themed around neighborliness, friendship, and shared activities — simple icons/emoji that appear and disappear in a loop. The style follows the platform's Soviet-flavored visual identity. Alongside the animation, a live infographic shows real numbers from the platform (people online nearby, messages in the last hour).

## Screen elements

- Icon animation themed around neighborliness/friendship/shared activities — appearing and disappearing.
- Live platform activity numbers, shown together with the icons.
- Centered on the screen — a birth year selector (year only, no day or month).
- A "next" button — icon only, no text label (e.g., an arrow).

## Logic

- Birth year is required to proceed.
- After picking a year and tapping the button — moves to the name entry screen.

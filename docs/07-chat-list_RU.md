# Экран 7 — Список чатов

## Назначение

Обзорный экран всех активных мэтчей/чатов пользователя — открывается, когда мэтчей больше одного.

## Элементы экрана

- Список чатов: один пункт на каждый мэтч.
- Переход в конкретный чат по тапу на пункт списка.

## Логика

- Новый мэтч добавляет новый пункт в список, не заменяя предыдущие чаты.
- Список актуален, пока активны сами чаты (см. экран 8 — время жизни чата).

## Открытые вопросы

- Что именно показывается в пункте списка (имя, превью последнего сообщения, непрочитанные) не определено.
- Сортировка списка (по времени последнего сообщения, по времени мэтча) не определена.

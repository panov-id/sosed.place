# Экран 4 — Публикация сообщения

## Назначение

Экран создания сообщения в ленте, открывается по кнопке "добавить".

## Элементы экрана

- Поле ввода текста — до 128 символов.
- Селектор количества человек — 1, 2, 3 ... N ("сколько вас").
- Слайдер радиуса места — размывает точную геопозицию автора: сообщение привязывается не к точным координатам, а к размытой зоне вокруг них (настройка приватности, а не видимости поста).
- Кнопка отправки.

## Логика

- Текст обязателен, количество человек и радиус размытия — настройки к посту.
- При отправке сообщение проходит ту же AI-проверку, что описана в разделе «Модерация» (Google Perspective API на токсичность + LLM-классификация тона: сексуальный подтекст, ЛГБТ-тематика) — публикация в ленту происходит только после прохождения проверки.
- Отправка также проходит через Cloudflare Turnstile (капча, в большинстве случаев невидимая для пользователя) и рейтлимит по IP через Bunny Shield. Обе проверки применяются только к публикации в ленту, не к отправке сообщений в чате.

## Открытые вопросы

- Остаётся ли отдельное поле город/страна из более раннего описания, или его заменил слайдер радиуса размытия.
- Значения по умолчанию и границы слайдера радиуса размытия не определены.
- Вид кнопки отправки (иконка/текст) не определён.

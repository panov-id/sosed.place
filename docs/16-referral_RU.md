# Экран 16 — Реферальная программа (за пределами альфы)

## Назначение

Механика приглашения новых пользователей с поощрением — часть внутреннего баланса (см. README, раздел «За пределами альфы»). Не входит в day0/альфу.

## Логика

- У пользователя есть персональная реферальная ссылка.
- Когда приглашённый заходит по ссылке и регистрируется, поощрение на баланс получают оба — и пригласивший, и приглашённый.

## Открытые вопросы

- Размер поощрения не определён.
- Экран/место, откуда пользователь берёт свою реферальную ссылку, не определено.
- Есть ли ограничение на количество приглашений — не определено.
- Момент начисления поощрения (сразу при регистрации приглашённого, или после какого-то условия) не определён.

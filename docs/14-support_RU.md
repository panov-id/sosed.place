# Экран 14 — Поддержка

## Назначение

Кнопка обращения в поддержку, доступная из приложения в любой момент.

## Элементы экрана

- Кнопка/иконка поддержки — всегда доступна.
- Поле ввода сообщения в поддержку.
- Кнопка отправки.

## Логика

- Отправленное сообщение попадает в таблицу в Supabase.
- При новом обращении срабатывает уведомление (email/webhook) для команды.
- Автоматической обработки/ответа нет — только фиксация обращения и уведомление.

## Открытые вопросы

- Откуда именно доступна кнопка (шапка ленты, настройки, отдельная иконка) не определено.
- Получает ли пользователь какое-то подтверждение отправки/ответ — не определено.

# Screen 12 — Sharing a Social Link in Chat

## Purpose

The ability to share the attached link (see screen 10) right inside a private chat.

## Screen elements

- A "share link" item in the chat's hidden menu — the same menu that holds report/block (see screen 5, applied to chat).
- A warning modal when tapping a link someone else sent — with "go" and "cancel" buttons.

## Logic

- Sharing the link is a one-time action for that specific chat, not turning on permanent visibility.
- The link is plain, not tied to a specific platform — the same field as in Settings (screen 10).
- Tapping a received link first shows a warning about leaving sosed.place for an external site, and only opens the link after confirmation.

## Open questions

- Whether the link can be re-sent or changed after already being sent in a given chat.

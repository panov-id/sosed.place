# Экран 3 — Лента

## Назначение

Основной экран приложения. Короткие сообщения от людей рядом, новые — внизу, как в чате, а не стопкой постов сверху. AI определяет язык каждого сообщения; по умолчанию около 95% того, что видно, — на языке пользователя, 5% — на других языках региона (доли настраиваются через переменную окружения).

## Элементы экрана

- Лента сообщений.
- Иконка-фильтр — открывает панель с фильтрами возраста и радиуса.

## Панель фильтров (по иконке)

- **Фильтр возраста** — слайдер диапазона. Для 18+ — плавный слайдер по любому диапазону, люди младше 18 не показываются никогда, независимо от настройки. Для младше 18 — слайдер тоже доступен, но с узким максимумом, не дающим увидеть взрослых.
- **Фильтр радиуса** — слайдер на карте: карта с визуальным кругом радиуса поверх неё, слайдером круг увеличивается/уменьшается.

## Логика

- Оба фильтра — часть интерфейса ленты, не отдельный шаг онбординга.
- Геолокация определяется автоматически при первом открытии ленты; при желании пользователь может задать её вручную через тот же круг на карте.

## Открытые вопросы

- Значения по умолчанию для радиуса (мин/макс/старт) не определены.
- Точный вид иконки-фильтра не определён.
- Экран/механика публикации сообщения (кнопка "добавить") пока не описаны — будут отдельным шагом.

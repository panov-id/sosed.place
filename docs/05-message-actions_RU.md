# Экран 5 — Действия с сообщением в ленте

## Назначение

Набор действий, доступных на карточке сообщения в ленте: лайк, жалоба, блокировка.

## Элементы карточки сообщения

- Иконка лайка — видна сразу на карточке, снаружи.
- Скрытое меню — содержит жалобу и блокировку, вызывается дополнительным действием (не отображается на карточке напрямую).

## Логика

- **Лайк.** Если лайк взаимный (оба участника лайкнули друг друга), между ними открывается приватный чат.
- **Жалоба.** Снижает квоту публикаций автора сообщения.
- **Блокировка.** Снижает квоту публикаций автора сообщения и дополнительно скрывает это сообщение из ленты лично для заблокировавшего — у остальных пользователей сообщение остаётся видимым.

## Открытые вопросы

- Скрывает ли жалоба сообщение лично для пожаловавшегося (как блокировка), или влияет только на квоту автора.
- Точный способ вызова скрытого меню (long-press, кнопка "...", свайп) не определён.
- Насколько сильно жалоба/блокировка снижают квоту не определено.

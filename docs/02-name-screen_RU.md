# Экран 2 — Имя

## Назначение

Второй шаг регистрации, сразу после выбора года рождения на экране загрузки — ввод отображаемого имени.

## Элементы экрана

- Поле ввода произвольного отображаемого имени — необязательно к заполнению.
- Кнопка перехода дальше — иконка без текстовой подписи (как на экране 1).

## Логика

- Имя необязательно: поле можно оставить пустым и продолжить.
- После нажатия кнопки — переход сразу в ленту.
- Отдельного экрана геолокации нет: определение позиции и настройка радиуса происходят внутри самой ленты, а не как отдельный шаг онбординга.

# Экран 17 — Стикеры (за пределами альфы)

## Назначение

Декоративные стикеры для сообщений/чата, покупаются за внутренний баланс (см. README, «За пределами альфы»). Не входит в day0/альфу.

## Логика

- Каталог стикеров фиксированный: пополняется и редактируется из админ-панели (см. `xor.ad/docs/panel_RU.md`), картинки хранятся через Supabase Storage/imgproxy.
- Пользователь покупает стикер за баланс в приложении и может использовать его в сообщении/чате.
- Обычные сообщения остаются текстовыми — стикер это отдельная, декоративная сущность, а не вложение произвольной картинки.

## Открытые вопросы

- Экран/место в приложении, откуда открывается каталог стикеров, не определено.
- Цены на стикеры не определены.
- Ограничение на количество стикеров в одном сообщении не определено.

# sosed.place
SOSED — русскоязычное/СНГ лицо гиперлокальной соседской сети. Знакомься с людьми в своём доме, офисе или квартале по совпадению настроения и интересов, а не бесконечным свайпам. Когда два соседа кликают друг с другом, открывается лёгкий live-чат — а дальше уже вы сами решаете, куда это ведёт. Только фронтенд — одно из двух лиц, работающих на общем бэкенде.
